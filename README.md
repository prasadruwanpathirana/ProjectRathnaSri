# ProjectRathnaSri
This is a charity project initiated by poem lovers

Steps to load the project

1. Install .NET Core 2.1
2. Install VS Code
3. Install GIT cli
4. Clone the project using below command 
    git clone https://github.com/prasadruwanpathirana/ProjectRathnaSri.git
5. Load the project folder using VS Code
6. Open the terminal and execute below command
    dontnet restore
    dotnet run
7. Open the below url and make sure the project is working correctly
    http://localhost:5000/
